Description

Project Overview

I created this project as an assignment for my Advanced CSS class at Weber State University. The purpose of this project was to create a new website in a landing page style. It was required to use css to create this site.

The images were provided by me or obtained from pexels.com.

Design & Build Process

After gathering the assets, I made two columns of content with a background image. The background image was chosen in relation to my introduction. The color overlay is complementary to the background colors. In coding the website, the website loading animations are large and noticeable to draw attention. The contact information is made simple with a button for email.

Problems Encountered

The image didn't display properly after setting the animation. This was fixed by setting the opacity to 1 to the end of the animation.
